# Criando-m-quinas-Virtuais-na-Azure
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO - para Azure AZ900 Criando máquinas Virtuais na Azure
Esse início rápido mostra como usar o portal do Azure para implantar uma máquina virtual (VM) no Azure que executa o Windows Server 2022 Datacenter. Para ver a VM em ação, você habilita o protocolo RDP na VM e instala o servidor Web do IIS.
